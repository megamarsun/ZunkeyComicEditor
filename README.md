# Zunkey Comic Editor (ズンコミ)

漫画やイラストの「セリフ」を簡単に入れ替えたり、描き文字（オノマトペ）を貼り付けたりできる編集ツールです。
Python製で、直感的な操作が特徴です。

## 📖 使い方

### 【1】 漫画画像を開く
1. 左上の **「1. ファイル操作」** > **[漫画を開く]** で画像（JPG/PNG）を読み込みます。

### 【2】 文字を消す（消しゴム）
1. **[スポイト]** ボタンを押し、画像内の背景色（白など）をクリックして色を取得します。
2. 消したい文字の上をなぞって消します。

### 【3】 文字を入れる
1. **「3. 文字入力」** にセリフを入力し、**[リストに登録]** を押します。
2. リスト内のセリフをクリックし、画像の好きな場所をクリックして配置します。

### 【4】 文字の調整
* 配置した文字をクリックして選択すると、**「5. テキスト設定」** でフォント、色、サイズ、縦書き、行間などを微調整できます。
* スライダー横の `+` `-` ボタンで細かく調整可能です。

### 【5】 描き文字（画像）を貼る
1. 右側の **[＋ 画像を追加]** で素材画像（PNG推奨）を読み込みます。
2. リストから選択し、画像上をクリックして配置します。
3. **「6. 画像設定」** で回転や拡大縮小が可能です。

### 【6】 保存
* **[画像書出 (PNG)]**: 1枚の画像として保存します。
* **[プロジェクト保存]**: 再編集可能な `.zmm` ファイルとして保存します。
